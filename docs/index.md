# **EPIC 1 — Today Dashboard**

### **Feature 1.1 — Daily Focus Card**

**User Story:**  
As a user, I want to see a single primary focus for the day so that I always know the most important thing to accomplish.

**Acceptance Criteria:**

- Shows 1 item pulled from weekly sprint tasks or WIG-related habits.
    
- Can mark the item as “Done.”
    
- Auto-refreshes to next priority after completion.
    

---

### **Feature 1.2 — Body & Food Panel**

**User Story:**  
As a bodybuilder, I want to log my meals and calories easily so I can stay on track with my 3,000 kcal and protein targets.

**Acceptance Criteria:**

- Displays calorie ring and macro bars.
    
- Quick-add buttons for common meals.
    
- Logging updates the day’s totals.
    
- Detects if I hit my calorie and protein targets.
    

---

### **Feature 1.3 — Deep Work Panel**

**User Story:**  
As a software engineer, I want to track my deep work hours so I stay consistent with focused coding time.

**Acceptance Criteria:**

- Start/stop timer for 25 or 50-minute sessions.
    
- Shows daily target (2 hours) and progress.
    
- Completed sessions add to weekly total.
    

---

### **Feature 1.4 — Daily Habits Panel**

**User Story:**  
As a user, I want to track key habits (protein, workout, reset, social) so I can build streaks that keep me motivated.

**Acceptance Criteria:**

- Each habit is tap-to-complete.
    
- Displays streak length.
    
- Streak breaks if not completed that day.

# **EPIC 2 — Weekly Sprint (Agile System)**

### **Feature 2.1 — Sprint Board (Kanban)**

**User Story:**  
As a user, I want to organize my tasks into backlog, this week, in-progress, and done so that I can manage my weekly commitments visually.

**Acceptance Criteria:**

- Draggable cards between columns.
    
- Columns: Backlog → This Week → In Progress → Done.
    
- Tasks retain area tags (Body, Work, Home, People).
    

---

### **Feature 2.2 — Weekly Sprint Planning**

**User Story:**  
As a user, I want to review and select tasks for my upcoming week so I can plan sprints aligned with my goals.

**Acceptance Criteria:**

- Ability to “Add to This Week” from backlog.
    
- Suggests tasks based on WIGs or unfinished tasks.
    
- Shows weekly commitment count.
    

---

### **Feature 2.3 — Weekly Retro**

**User Story:**  
As a user, I want a simple weekly reflection so I can adjust my habits and improve next week.

**Acceptance Criteria:**

- Quick rating sliders for Health, Work, Home, Relationships.
    
- Notes box for insights.
    
- Summary saved to history.

# **EPIC 3 — Scoreboard (4DX Metrics)**

**User Story:**  
As a user, I want to see my lead measures (calories, protein, workouts, deep work) so I can track the behaviors that drive results.

**Acceptance Criteria:**

- Weekly bars or line charts.
    
- Metrics: meals logged, calories, protein, deep work minutes, workouts.
    
- Weekly target markers visible.
    

---

As a user, I want to track lag outcomes such as weight or PRs so I can see progress from my consistent behaviors.

**Acceptance Criteria:**

- Trend lines for: bodyweight, bench, squat, deadlift, OHP (optional).

**Acceptance Criteria:**

# **EPIC 4 — Backlog & Goal Alignment (OKRs + GTD)**

### **Feature 4.1 — OKR Management**

**User Story:**  
As a user, I want to define quarterly Objectives and Key Results so my weekly tasks align with long-term goals.

**Acceptance Criteria:**

- Create/edit/delete OKRs.
    
- Objectives grouped by: Body, Work, Home, Relationships.
    
- Key Results link to tasks + habits.
    

---

### **Feature 4.2 — Backlog System**

**User Story:**  
As a user, I want a place to quickly dump tasks into a backlog so I can sort them later without losing ideas.

**Acceptance Criteria:**

- Fast “+ Task” creation.
    
- Tags available: Body, Work, Home, People, Admin.
    
- Sort/filter/search functions.
    

---

### **Feature 4.3 — Habit Library**

**User Story:**  
As a user, I want to choose which habits I’m focusing on right now so my daily view stays uncluttered.

**Acceptance Criteria:**

- Add, remove, or hide habits.
    
- Habits categorized by life area.
    
- Toggling affects Today Dashboard.
